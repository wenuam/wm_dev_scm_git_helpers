# wm_dev_scm_git_helpers

Some helper scripts and tips

* Home

https://github.com/wenuam/wm_dev_scm_git_helpers

## Scripts

 Script				| Action
 :--				| :--
`admin_keymgr`		| Open certificate management dialog
`git_xxx_all.bat`	| Perform the `xxx` command on all found cloned repositories (rename it if needed)
`git_push_loop.bat`	| Loop pushing until the **local** repository is synced (copy it where needed)
`git_set_token.bat`	| Apply new access 'ghp_xxx' token to all found cloned repositories (as parameter or asked)
`subf_on_date`		| Sort files into created subfolders based on files' date (opt. filter on ext as parameter)
